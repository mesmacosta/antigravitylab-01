# Getting Started with the Antigravity Ecosystem

Welcome to the **Getting Started with the Antigravity Ecosystem**! This repository contains a comprehensive Google Codelab that teaches developers how to build an event-driven, secure, AI-powered application on Google Cloud using the **Antigravity Agentic Framework**.

Instead of writing code manually, you will orchestrate a team of specialized AI agents (Developer, Architect, Data Engineer, and SRE) to provision infrastructure, write microservices, and configure CI/CD pipelines.

---

## 📂 Repository Structure

This repository acts as both the **source code** for the codelab tutorial and the **working directory** where the agents will generate code during the lab.

| Directory | Purpose |
|-----------|---------|
| `.agents/` | **The Agent Intelligence**. Contains the AI persona definitions (`agents.md`), the slash-command workflows (e.g., `/developer`, `/sre`), and the highly-constrained `skills/` that provide the exact `gcloud` templates and validation scripts the agents must follow. |
| `codelabs/` | **The Codelab Source**. Contains the master Markdown file (`getting-started-with-antigravity-ecosystem.md`) and the generated static HTML site (`getting-started-with-antigravity-ecosystem/`). |
| `docs/` | **Development Documentation**. Contains `DEVELOPMENT.md` which explains how to author skills, edit workflows, and rebuild the codelab HTML. |
| `app/` | **(Generated during lab)**. The Flask microservice code scaffolded by the Developer agent in Lab 1. |
| `infra/` | **(Generated during lab)**. The Cloud Build CI/CD configurations generated by the SRE agent in Lab 4. |

## 🚀 How to Take the Codelab

If you just want to take the lab and learn about Antigravity, you can serve the generated HTML locally:

1. Ensure you have [Go](https://go.dev/doc/install) installed.
2. Install the Codelabs Command Line Tool (`claat`):
   ```bash
   go install github.com/googlecodelabs/tools/claat@latest
   export PATH=$PATH:$(go env GOPATH)/bin
   ```
3. Serve the static site:
   ```bash
   cd codelabs
   claat serve -addr localhost:9090
   ```
4. Open [http://localhost:9090/getting-started-with-antigravity-ecosystem/](http://localhost:9090/getting-started-with-antigravity-ecosystem/) in your browser.

## 🛠️ How the Codelab is Generated

The beautiful, interactive codelab UI is generated from a single markdown file using Google's open-source **`claat`** tool.

If you want to contribute to the lab or modify the tutorial instructions:

1. Edit the source markdown file at `codelabs/getting-started-with-antigravity-ecosystem.md`.
2. Follow the Google Codelabs formatting rules (e.g., using `Positive` and `Negative` blocks for callouts). 
3. Regenerate the HTML assets:
   ```bash
   cd codelabs
   claat export getting-started-with-antigravity-ecosystem.md
   ```
4. Commit your changes to both the markdown source and the generated HTML files.

For a deeper dive into editing agent skills and workflows, please see [docs/DEVELOPMENT.md](docs/DEVELOPMENT.md).
